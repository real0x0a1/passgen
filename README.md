## Setup

1. Install:
```bash
   make install
```
2. Run the `passgen`:
```bash
  passgen
```
3. Uninstall :
```bash
   make uninstall
```

## Usage
```bash
   passgen <from-len> <to-len> [charset] [-t [FIXED]@@@@] [-s startblock] -o <output_file>
```
